# 📄 CAHIER DES CHARGES — ScanAR (Voro) (version v0)

---

# 🧠 1. Vision produit

## 🎯 Objectif

Créer une plateforme SaaS permettant de :

* transformer une image en modèle 3D (IA)
* visualiser ce modèle en réalité augmentée (AR)
* partager via lien ou QR code
* démocratiser l’AR pour tous les objets du quotidien

---

## 🌍 Positionnement

Produit universel AR :

* e-commerce (produits)
* restauration (plats)
* particuliers (objets perso)
* entreprises (catalogues interactifs)
* marketing (QR code AR publicitaire)

👉 Concept clé :

> “Tout objet peut devenir une expérience AR accessible via un simple scan”

---

# 🤖 2. IA / génération 3D

## 🔹 Pipeline IA (Hunyuan3D v1)

### Étapes backend :

1. Upload image utilisateur
2. Envoi vers worker Python (GPU)
3. Hunyuan3D génère :

   * mesh 3D
   * textures
4. Post-processing automatique :

   * clean mesh
   * optimisation topology
   * compression GLB
5. Export :

   * `.glb` (web + Android)
   * `.usdz` (Apple AR QuickLook)

---

## ⚙️ Post-processing

* automatique par défaut
* option “amélioration qualité” côté user
* objectif : fidélité maximale à l’image

👉 IMPORTANT :

* simplification mesh (decimation)
* texture baking optimisé
* compression Draco

# 🧱 3. Architecture technique

## 🏗 Stack globale

### Frontend

* Next.js (App Router)
* Three.js / model-viewer

### Backend

* Next.js API routes (léger)
* FastAPI (Python GPU service)

### IA Worker

* Python + Hunyuan3D
* Docker container GPU

### Communication

* Redis Queue (jobs async)
* WebSocket (progress génération en temps réel)
