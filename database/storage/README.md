# Storage Buckets

Configuration Supabase Storage.

**Contenu**:
- `buckets_setup.sql` - Créer buckets (images, models-glb, models-usdz)
- `storage_policies.sql` - RLS pour storage access

**Buckets à créer**:
- `images` - Images originales utilisateurs
- `models-glb` - Modèles 3D .glb (web/Android)
- `models-usdz` - Modèles 3D .usdz (iOS AR)
