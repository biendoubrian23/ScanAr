# Migrations SQL

Scripts de création des tables Supabase (exécutés une fois, une seule fois).

**Format**: `XXX_description.sql`

**Exemples**:
- `001_tables_users.sql` - Créer table users
- `002_tables_models.sql` - Créer table models_3d
- `003_tables_ar_links.sql` - Créer table ar_links
- `004_tables_analytics.sql` - Créer table analytics

⚠️ À chaque création de nouvelle table → 1 fichier SQL
