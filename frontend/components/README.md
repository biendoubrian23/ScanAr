# Components Réutilisables

Composants React pour frontend.

**Structure**:
- `ui/` - Composants génériques (Button, Input, Modal, etc)
- `layout/` - Header, Navbar, Sidebar, Footer
- `viewers/` - 3D Viewer, AR Viewer
- `forms/` - Form components (upload, settings)
- `cards/` - Model cards, Link cards
- `modals/` - Modales (confirmations, etc)

Chaque composant:
- ✅ Réutilisable
- ✅ TypeScript
- ✅ Tailwind styled
